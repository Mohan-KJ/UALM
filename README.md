# UALM
UALM is a high fidelity Streamlit app for full spectrum audio intelligence. Powered by Whisper base, fine-tuned AST, and fine-tuned DistilBERT, it handles multilingual ASR, translation, and emotion mapping. It features a custom MFCC-based diarization engine for speaker timelines and an AST gated VAD to intelligently skip speech logic for noise.
